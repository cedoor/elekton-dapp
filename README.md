## Install

With the following installed:
- git
- node >= 12
- npm >= 6

Clone the repo and install the dependencies from npm.

```bash
git clone https://github.com/elekton/elekton-dapp.git
cd dapp
npm i
```

## Usage

For local *development* with expo:

```bash
npm start
```
